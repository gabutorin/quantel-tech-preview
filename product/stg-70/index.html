STG‑70A/H · Волоконно‑оптические гироскопы (ВОГ)

ВОГ трёхосный STG‑70 (разнесённый)

3 осиразнесённыйуход ≤0.002°/√ч (H)±300°/с≥200 Гц

Трёхосный ВОГ разнесённого исполнения для инерциальных измерительных систем. Высокая надёжность, гибкая компоновка под платформу заказчика.

Применение: бортовые ИНС, курсовертикали, гирокомпасы. Исполнения A и H — по возрастанию точности.

Общие параметры серии: питание +5 В, время готовности ≤ 3 с (выход на номинал без прогрева), рабочая температура −45…+70 °C, хранение −55…+85 °C. Полностью твердотельный цифровой ВОГ с замкнутым контуром. Полный даташит с габаритным чертежом, вибро/ударными данными и картой интерфейса — по запросу.
